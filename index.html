ideh.v6@cit.inpt: ~ $ ideh --help

All commands available are here: ideh [COMMAND] [OPTION]

=====> COMMAND:

init: initialize your candidature to take part in the next edition of IDEH

add -fn -ln -email: add a team member

push: send your candidature to preview

=====> OPTIONS:

--name: Give a name to your team; if not mentioned, you will be attributed a name as "Team__number"

--help: Show this help panel

-fn: First name

-ln: Last name

-email: Email

--leader: To indicate the leader of the team

--school: When you try to add a member, you should mention the school

========> Example:

add -fn John -ln Doe --school INPT -email john.doe@mail.mail --leader

NB:
- You are allowed to have one leader, and make sure all provided information is correct; if not, you may be disqualified automatically.
- You are allowed to add four members; one of them must be the leader. If not mentioned, it will automatically be attributed to the first added person.
- ideh push works only and if only there are four members.
- If there is a problem with your candidature, please let us know {*_*}.
- Start now by running: ideh init
